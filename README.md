# Hateoas Escape Room

Minimalist Node.js API app using TypeScript.

## Features
- Express server
- TypeScript configuration
- Minimal dependencies

## Scripts
- `npm run build` — Compile TypeScript
- `npm start` — Run compiled server

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the project:
   ```bash
   npm run build
   ```
3. Start the server:
   ```bash
   npm start
   ```

The API will be available at `http://localhost:3000/`.
